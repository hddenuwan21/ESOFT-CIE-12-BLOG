    How To Activate Windows 10 Using Cmd?

    Step 1: run administrator cmd.

    Step 2: Use the command “slmgr /ipk yourlicensekey” to install a license key. Replace “yourlicensekey” with any of the factory license keys below. Make sure to choose the right one for your Windows version.

    Default key:

    Hit [Enter] key to execute the command. Wait for a few seconds, and a prompt should appear..

    Step 4: 5. Use the command “slmgr /skms kms8.msguides.com” to connect to the msguides independent free KMS server.
